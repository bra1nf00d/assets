----Выгрузка библиотек----

JS
1. file.js кидать в /js/vendor/
2. Подключить file.js -в- script.js

CSS
1. file.css переделать под .scss
2. file.scss кидать в /scss/vendor/
3. Подключить file.scss -в- _vendor.scss

----------------------------


-------Выгрузка иконок-------

ICON
1. Создать, загрузить все иконки в папку icon
2. Папку icon переместить в resource
3. Подключать иконки уже из папки icon
../icon/file.svg

FONT-ICON
1. Картинки конвертировать в иконочный шрифт
https://icomoon.io/app/#/select
2. Создать, загрузить в папку icon
3. Переместить в resource
4. (Для min версии) JS/CSS загрузить в script.js || _vendor.scss (только .min.css)
5. (Для обычной) JS/CSS загрузить в vendor.js || _vendor.scss

-------------------------------

-------------Шрифты------------

FONT
1. Выгружать шрифты только в формате: .ttf
2. Перед выгрузкой, переименовать в Font-Weight (Font - имя шрифта, Weight - Жирность)

-------------------------------

------------Спрайты------------

SPRITE
1. Спрайт создается, когда file.svg лежит в папке /img/svg/
2. Чтобы подключить спрайт, нужно вставить:
<svg class="">
<use xlink:href="../img/svg/sprite.svg#УКАЗАТЬ_ИМЯ_SVG"></use>
</svg>
